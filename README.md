# Benportfolio

AI-enabled Functional Analyst portfolio built on top of the
[Creative Tim Next.js + Tailwind Portfolio Page](https://github.com/creativetimofficial/nextjs-tailwind-portfolio-page).

## Tech Stack

- Next.js
- React
- Tailwind CSS
- TypeScript (if present in template)
- Framer Motion (if present in template)
- Deployed on Vercel (planned)

## Purpose

This site showcases:

- My profile as a Functional Analyst focused on AI-enabled operations
- Selected case studies (HRIS, workflow automation, AI+Ops)
- Resume and contact information

# Install dependencies
